# Raw Sources

Drop source documents here. These are **immutable** — the LLM reads them but never modifies them.

## How to ingest a new source

1. Copy the file here (PDF, markdown, text, YAML, image)
2. Tell Claude: `ingest [filename]` or `ingest raw/[filename]`
3. Claude will process it per the SCHEMA.md workflow
